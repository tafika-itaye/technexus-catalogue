## 🚀 Deploy to GitHub Pages in 3 Steps

```bash
git init && git add . && git commit -m "TechNexus catalogue March 2026"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/technexus-catalogue.git
git push -u origin main
```

Then: **Repository → Settings → Pages → Source: main / root → Save**

Your catalogue goes live at: `https://YOUR_USERNAME.github.io/technexus-catalogue/`
